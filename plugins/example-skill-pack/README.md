# Example Skill Pack

A demonstration plugin showing the standard structure for Claude Code skill plugins.

## Skills

- **example-skill** - A simple greeting skill for testing plugin functionality

## Installation

```bash
claude plugin add <repo-url>/plugins/example-skill-pack
```

## Local Testing

```bash
claude --plugin-dir /path/to/plugins/example-skill-pack
```

## Usage

After installation, use `/example` to trigger the example skill.
